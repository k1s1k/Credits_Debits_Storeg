Credits_Debits_Storeg ( MoneyCRM ) - это простое приложение учёта личных кредитов и вкладов. 

*Все данные сохраняются локально на вашем устройстве

## 🚀 Функции
- 📈 Учёт вкладов с расчётом процентов
- 💳 Учёт займов
- 🧮 Автоматический расчёт дохода по формуле простых процентов
- 💾 Хранение данных в localStorage
- ✏️ Редактирование и удаление записей

## 🛠️ Технологии
- React 18
- SCSS
- localStorage

## 📦 Установка
```bash
git clone https://github.com/k1s1k/Credits_Debits_Storeg.git
cd Credits_Debits_Storeg
npm install
npm run dev
